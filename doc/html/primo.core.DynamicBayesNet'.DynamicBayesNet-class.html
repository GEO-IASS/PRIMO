                    DynamicBayesNet

This is the implementation of a dynamic Bayesian network (also called temporal Bayesian network).

Definition: DBN is a pair (B0, TwoTBN), where B0 is a BN over X(0), representing the initial distribution over states, and TwoTBN is a 2-TBN for the process. See Koller, Friedman - "Probabilistic Graphical Models" (p. 204)

Properties: Markov property, stationary, directed, discrete, acyclic (within a slice)
