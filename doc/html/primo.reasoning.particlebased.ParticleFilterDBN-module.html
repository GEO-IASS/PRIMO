particle_filtering_DBN(network, N, T, get_evidence_function, interval=0)

source code 

Create N samples for the given network with T time slices.

Keyword arguments: network -- a DynamicBayesNet N -- number of samples T -- number of time slices (set -1 for infinite loop) get_evidence_function -- a function that returns a dict with the following structure: {node1:evidence1, node2:evidence2, ...} interval -- time to sleep between two sample loops (only if T = -1)

Returns a dict of N samples
